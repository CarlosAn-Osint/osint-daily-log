# OSINT Daily Log

Registro diario de investigación OSINT, análisis geopolítico y observación de dinámicas informacionales.  
Este repositorio documenta procesos, fuentes, métodos y reflexiones analíticas.

## Estructura
- Anotaciones diarias de investigación
- Enlaces relevantes y archivados
- Mapas de actores e influencias
- Síntesis y conclusiones

## Objetivo
Desarrollar un marco personal de análisis riguroso y sistemático.
